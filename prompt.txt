Build a 100% pixel-perfect Instagram frontend clone using React.js. Every single page that exists on Instagram must be built. No placeholders, no TODOs, every screen complete.

## Tech Stack
- React.js with Vite
- React Router DOM v6
- Tailwind CSS
- Axios + React Query
- Lucide React (icons)
- Framer Motion (animations)
- Images: https://picsum.photos/600/600?random={id}
- Avatars: https://i.pravatar.cc/150?img={id}

## INTERACTIONS & MICRO-ANIMATIONS

- Double-tap image → big red heart bursts from tap point (scale 0 → 1.2 → 0)
- Like button → outline to filled red with bounce (scale 1 → 1.3 → 1)
- Bookmark → outline to filled black with pop
- Follow button → "Follow" (blue bg) → "Following" (white bg, grey border)
- Story ring → gradient when unseen, grey #DBDBDB when seen
- Notification badge → red dot (8px) on messages/notifications icon
- Post carousel → swipe or arrow, dot indicators at bottom
- Reel heart double-tap → burst of hearts float up
- Stories → tap right 70% = next, left 30% = prev, hold = pause
- Chat → typing indicator (three bouncing dots)
- Loading → skeleton screens (grey shimmer animation) for posts, stories, profiles
- Toast notifications → slide up from bottom (green for success)
- Modal → fade in + scale from 0.95 to 1

## RESPONSIVE BREAKPOINTS

Mobile (< 768px):
- No sidebar
- Bottom nav with 5 icons (Home, Search, Reels, Shop, Profile)
- Full width posts
- Modals become full screen bottom sheets
- No right panel

Tablet (768px - 1024px):
- Icon-only sidebar (72px)
- No right panel
- Wider posts

Desktop (1024px - 1280px):
- Icon-only sidebar (72px)
- Right panel visible

Large Desktop (> 1280px):
- Full sidebar with labels (244px)
- Full right panel
